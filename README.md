# Release Lyntrax

Repositorio oficial de distribución de versiones de **Lyntrax**.

## 📦 Descripción

Este repositorio está destinado exclusivamente a la publicación y almacenamiento de las versiones oficiales (Releases) de Lyntrax.

Aquí encontrarás los paquetes de instalación y archivos ejecutables correspondientes a cada versión publicada.

## 🚀 Descargas

Para descargar la última versión disponible, visita la sección de **Releases** del repositorio.

Cada versión incluye:

* Instalador o ejecutable.
* Notas de la versión (Release Notes).
* Correcciones de errores.
* Nuevas funcionalidades.
* Mejoras de rendimiento y estabilidad.

## 📋 Versionado

Lyntrax utiliza versionado semántico:

```text
MAJOR.MINOR.PATCH

Ejemplo:
1.0.0
1.1.0
1.1.1
```

* **MAJOR:** cambios incompatibles o grandes actualizaciones.
* **MINOR:** nuevas funcionalidades compatibles.
* **PATCH:** correcciones de errores y mejoras menores.

## 📝 Notas Importantes

* Este repositorio no contiene el código fuente del proyecto.
* Su propósito es servir como punto central de distribución de versiones oficiales.
* Se recomienda descargar únicamente archivos publicados en las Releases oficiales.
